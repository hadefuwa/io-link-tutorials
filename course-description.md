# IO-Link Practical Course — playlist description (copy-paste ready)

Learn IO-Link the practical way — real hardware, real screens, no fluff.

IO-Link is quietly changing industrial sensing: it turns simple on/off sensors into smart devices that stream live data, diagnostics and configuration over a single standard cable. This hands-on course builds you up from "what even is IO-Link?" all the way to reading and controlling devices from a browser, a PLC, and your own custom apps.

No endless theory and no slideshows. Every episode is done live on a real bench — an ifm AL1350 master and a mix of sensors — so you can follow along and actually do it yourself.

🔧 WHAT YOU'LL LEARN
• What IO-Link is, and why it beats a plain digital sensor
• Configuring smart devices with manufacturer software (ifm moneo)
• Getting data straight out of the master — the built-in web server and the IoT Core API
• IODD files: how software understands any sensor from any brand
• Reading & writing IO-Link data from a Siemens S7-1200 PLC — with NO Profinet — over HTTP and MQTT
• Building your own live dashboards on top of it all
